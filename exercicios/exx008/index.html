Principais formatações

Negrito / destaque

Nesta frase, foi utilizado a formatação em negrito(b), sendo assim foi feito com a tag <b>(forma não semântica).

Nesta frase, foi utilizada a formatação em destaque(strong), sendo assim foi utilizada a tag <strong>(semântico).

Ítalico / Ênfase

Nesta frase, foi utilizada a formatação ítalico(i) sendo assim foi utilizada a tag <i>(forma não semântica)

Nesta frase, foi utilizada a formatação ênfase(em) sendo assim foi utilizada a tag <em>(forma semântica)

Texto marcado

podemos marcar um texto com a tag <mark>

Textos pequenos

Podemos fazer um texto pequeno com a tag <small>(semântica)

Texto deletado e Texto inserido

Com a tag <del> podemos fazer um texto deletado: Lorem ipsum dolor sit, amet consectetur adipisicing elit. Impedit architecto atque, nesciunt tempora nulla error et vero eligendi voluptate obcaecati quaerat rerum totam, accusantium minus sed porro neque assumenda numquam?

Com a tag <ins> podemos fazer um texto inserido: Lorem ipsum dolor sit amet consectetur adipisicing elit. Dignissimos ipsa nobis totam, facilis fugit minima explicabo pariatur dolores quod atque excepturi repudiandae vitae, inventore id, error eos at nesciunt sapiente?

Texto sobrescrito e Texto subscrito

Podemos fazer um texto sobrescrito x20+10 com a tag <sup>

Podemos fazer um texto subscrito H20 com a tag <sub>
